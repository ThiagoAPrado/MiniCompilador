1.  Declaração simples de função (válido) 
    funcao soma(a, b) = a + b

2.  Função sem parâmetros (válido) 
    funcao x() = 10 
    x()

3.  Parâmetro duplicado (erro semântico) 
    funcao teste(a, a) = a + 1

4.  Funções com mesmo nome (erro semântico) 
    funcao x(a) = a + 1 
    funcao x(b) = b + 2

5.  Chamada com argumentos demais (erro semântico) 
    funcao soma(a, b) = a + b 
    soma(1, 2, 3)

6.  Chamada com argumentos de menos (erro semântico) 
    funcao soma(a, b) = a + b 
    soma(10)

7.  Chamada de função não declarada 
    x(10)

8.  Expressão válida usando chamada de função 
    funcao dobro(a) = a * 2
    funcao soma(a, b) = a + b 
    x = soma(10, dobro(3))

9.  Variável não declarada (erro semântico) 
    funcao f(a) = a + x

10. Variável declarada por atribuição (válido) 
    x = 10 
    y = x + 5

11. Uso de variável antes da atribuição (erro semântico) 
    y = x + 2

12. Recursão direta (válido!) 
    funcao fat(n) = n * fat(n - 1)

13. Recursão indireta (válido!) 
    funcao a(x) = b(x) 
    funcao b(x) = a(x)

14. Função mais complexa 
    funcao calc(a, b, c) = (a + b) * c / (a - b)
    x = calc(10, 2, 3)

15. Expressões com potência 
    x = 2 ^ 3 ^ 2

16. Declaração seguida de chamada complexa 
    funcao soma(a, b) = a + b
    funcao mult(a, b) = a * b 
    x = soma(3, mult(2, soma(1, 1)))

17. Função usando parâmetros corretamente 
    funcao teste(a, b, c) = a + b * c 
    teste(2, 3, 4)

18. Token inválido 
    x = 10 @ 20

19. Parênteses aninhados 
    x = (1 + (2 * (3 + 4)))

20. Atribuição após função 
    funcao soma(a, b) = a + b 
    y = soma(4, 5)
